## Установка и запуск

   Склонируйте репозиторий:
   ```git clone git@github.com:Ostashev/mushrooms.git```

   Перейдите в дирректорию:
   ```cd mushrooms```

   Запустите контейнеры:
   ```docker-compose up --build```

   После успешного запуска, сервер API будет доступен по адресу:
   ```http://localhost:8000```
